# Codex Agent Routing — GMTK26

Codex agents are on-demand collaborators. This project provides named custom
profiles in `.codex/agents/`; use one when delegating a matching task. The role
cards in `agents/` remain their fuller working briefs. The root `AGENTS.md`
automatically gives Codex the shared-memory rules.

For delegated work, name the role, include the current `gamejam/` state, give one
bounded deliverable, and require a concise handoff. The recommended routing is:

- **Design Brain:** divergent theme responses, then a concrete prototype test.
- **Scope Producer:** feasibility review before committing to a direction.
- **Godot Engineer:** implement the chosen vertical slice in small testable steps.
- **Art and UX Partner:** readability and feedback plan once the loop is defined.
- **Playtest Critic:** evaluate an actual build, not a pitch.

Use one specialist at a time unless parallel work is truly independent. The
orchestrator resolves recommendations into `gamejam/DECISIONS.md`.
